## Insertion Sort (Descending Order)

This Python file implements the Insertion Sort algorithm
from Cormen et al. but modified to sort in decreasing order.

### How to run

python insertion_sort_desc.py


### Output Example

Original array: [5, 2, 9, 1, 5, 6]  
Sorted array (decreasing): [9, 6, 5, 5, 2, 1]
